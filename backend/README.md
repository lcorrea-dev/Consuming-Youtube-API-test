# Backend app

Developed in Node.js v14.16.1

## Environment setup

1. Write your own .env file in the root folder following .env.example

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and start the app

```
npm run start
```

### Run your tests

```
npm run test
```
